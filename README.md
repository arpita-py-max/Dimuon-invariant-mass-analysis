# Dimuon-invariant-mass-analysis
Dimuon invariant mass analysis using CERN CMS open data with python and gaussian fitting.
This project reconstructs the Z boson resonance from real CMS collision data using Python.

Objective

Identify the Z boson from dimuon decay channels using invariant mass analysis.

Methods

- Data cleaning and preprocessing
- Opposite-sign charge event filtering
- Invariant mass reconstruction
- Histogram visualization
- Gaussian fitting using least squares

Results

Measured Z boson peak near 90.82 GeV with detector resolution sigma ≈ 2.21 GeV.

Tools Used

Python, Pandas, NumPy, Matplotlib, SciPy
